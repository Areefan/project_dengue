## การรันโปรแกรม

### เรียกใช้คำสั่ง Docker

`docker-compose up -d `

### ตรวจสอบว่า container ถูกสร้างสำเร็จ
`docker ps`

### เข้าใช้งาน phpMyAdmin
`http://localhost:8080`

เข้าใช้งาน r506-db ให้ระบุดังนี้

` Server: r506-db
Username: user
Password: user
`
### หรือ เช้าใช้ผ่าน NAVICAT

` 
Host: localhost
PORT: 6001
Username: user
Password: user
`
